### What Karpathy's Autoresearch Is
- Autonomous agent loop for ML training: agent edits code, runs timed experiments, commits only changes that improve a single measurable metric
- Key quote: "any metric you care about that's reasonably efficient to evaluate can be autoresearched by an agent swarm"

### Atlas-GIC (Chris Worsey's Implementation)
- Repo: github.com/chrisworsey55/atlas-gic
- Built by General Intelligence Capital
- Karpathy's autoresearch loop + Soros's reflexivity + MiroFish swarm simulation applied to financial markets
- Agent prompts = weights. Sharpe ratio = loss function. Git = version control. No GPU needed ($20/mo VM).
- 25+ agents across 4 layers debate markets daily
- Architecture:
  - Layer 1 (Macro, 10 agents): central bank, geopolitical, China, dollar, yield curve, commodities, volatility, EM, news sentiment, institutional flow
  - Layer 2 (Sector Desks, 7 agents): semiconductor, energy, biotech, consumer, industrials, financials, relationship mapper
  - Layer 3 (Stock Pickers): Druckenmiller, Aschenbrenner, Baker, Ackman, CRO, Alpha Discovery, Autonomous Execution personas
  - Layer 4 (CIO): synthesizes all layers, weighted by Darwinian agent scores
- Worst agent by rolling Sharpe gets prompt rewrite → 5-day test → keep or revert
- Results: 378 trading days, +22% deployed, AVGO called at $152 → +128%
- Self-spawning: detected knowledge gaps autonomously, grew from 25 to 31 agents over 6 months
- Regime cohorts: trained separate agent cohorts on distinct market regimes (bull, crisis, rate tightening, euphoria, recovery)
- JANUS: meta-layer above cohorts, weights them by recent accuracy — emergent regime detector
- Reflexive simulation: 5 feedback loops modeled (price→fundamentals, P&L→behavior, narrative→flows, market→policy, reflexive reversal detection)
